# AGOSF_Player_MP3
Un proyecto con HTML el cual es, basicamente, un reproductor de MP3

Puedes modificarlo, cambiarle alguna que otra cosa, o incluso, poder usar este de base para hacer tu propio reproductor
de musica, aunque si lo usaras de soporte puedes almenos darme creditos.

ADVERTENCIA: El codigo del HTML Y JS Fue hecho con IA, esto debido a que no soy un programador experimentado, pero igual por cada
versión que haga añadire mas opciones sin necesidad de IA.
